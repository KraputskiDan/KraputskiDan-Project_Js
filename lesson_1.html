Nouns Singular in the Genitive Сase

In this lesson you will begin studying the Genitive Case (Без паники! Don’t panic!) Key phrases: У меня нет времени, нет денег. Я люблю чай без сахара и без лимона.

Genitive endings vary according to noun’s gender
  • У меня есть + что?
  • (Nominative case)
  • У меня нет + чего?
  • (Genitive case)
она
  • У меня есть вилка.
  • У тебя есть тарелка.
  • У вас есть идея.
  • У них есть проблема.
  • У тебя нет вилки.
  • У него нет тарелки.
  • У вас нет идеи.
  • У них нет проблемы.
Ending: -а → -ы, -я → -и, -ь → -и -а → -и
он/оно
  • У меня есть стол.
  • У тебя есть заказ.
  • У него есть чай.
  • У вас есть конкурент.
  • У нас есть пиво.
  • У меня нет стола.
  • У тебя нет заказа.
  • У него нет чая.
  • У вас нет конкурента.
  • У нас нет пива.
Ending: zero ending → -а, zero ending → -x, -ь → -я, -о → -а, -е → -я
  • Remember
  • У меня есть время. — У меня нет времени.
  • У меня есть деньги. — У меня нет денег.
  • У меня есть кофе. — У меня нет кофе.

Negative construction
  • У меня не было ресторана.
  • У меня не было работы.
  • У меня не было вина.
  • У меня не было денег.
  • У меня нет ресторана.
  • У меня нет работы.
  • У меня нет вина.
  • У меня нет денег.
  • У меня не будет ресторанa.
  • У меня не будет работы.
  • У меня не будет вина.
  • У меня не будет денег.
  • Remember
  • У меня никогда не было работы. (double negation + Genitive)
  • У меня всегда была работа.
